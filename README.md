# Fonio Track - Hack Vienna Hackathon

## Showcase features

- Book appointments, cancel appointments
- Legal verification when booking a call that call back is permitted 
- When cancelling, the user is first tried to be converted (offer 2 other spots)
- Cancel appointment, this triggers the booking loop (trying different users based on their rank on the waitlist)
- We engineered the waitlist based on factors
- You can ask general questions about appointment price, type, etc.
- Complete webui including conversation and voice recording
- Reroute to a human if the AI cannot assist further in edge cases
- It can call back later based on time specified ("call me in 1 minute")
